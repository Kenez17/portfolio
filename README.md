# Portfolio de Automatización
**Kevin Martínez** — Automatización n8n, Agentes IA, Trading Bots, Integraciones CRM

## 🚀 Cómo usar
1. Clona este repo
2. Abre `index.html` en tu navegador
3. Para producción: despliega con Vercel o Netlify (conectado a este repo)

## 📹 Agregar videos
Los placeholders `video-placeholder` en `index.html` están listos para reemplazar con embeds de YouTube.
Busca `subir video a YouTube` en el HTML y cámbialo por el iframe de tu video.
